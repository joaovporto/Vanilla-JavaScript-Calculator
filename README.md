## 🚀 Installation

```
$ git clone https://github.com/joaovporto/Vanilla-JavaScript-Calculator.git
$ cd Vanilla-JavaScript-Calculator
$ npm install
$ npm start
```

The application will open in the browser on http://localhost:3000
